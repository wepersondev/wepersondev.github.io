# CSS

  • 클래스 네이밍은 BEM 방식을 사용한다.
  • ID 대신 클래스를 사용한다.
  • 태그 선택자 대신 클래스 선택자를 사용한다.
  • 가능한 한 축약형을 사용한다.

# Sass

  • Intend Style인 sass 문법을 사용한다.
  • 변수명은 케밥 케이스를 사용한다.
  • 믹스인은 함수로만 사용한다.
  • Extend 지시자는 사용하지 않는다.
  • 선택자 중첩은 최대 3단계까지만 사용한다.

각 실 사용 예시는 업데이트 예정입니다.
